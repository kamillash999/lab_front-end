# Лабораторная работа №11: Основы React Native и Expo

## Описание работы
Эта лабораторная работа посвящена изучению основ мобильной разработки. Вместо привычного веба я научилась создавать интерфейсы для смартфонов, используя нативные компоненты и систему навигации.

## Выполненные задания:

### 1. Core Components & Styling (Task 1)
В этой части я реализовала карточку профиля пользователя.
* **Что я использовала:**
    * `View` — в качестве основного контейнера (аналог div).
    * `Text` — для вывода всех текстовых данных.
    * `Image` — для отображения аватара пользователя.
    * `StyleSheet` — для создания стилей, включая закругление углов (`borderRadius`) и тени (`elevation`).
* **Чему я научилась:** Я разобралась с системой Flexbox в React Native и поняла, что по умолчанию элементы располагаются в колонку (`flexDirection: 'column'`).

### 2. Stack Navigation (Task 2)
Я реализовала простейшую навигацию между двумя экранами приложения.
* **Библиотеки:** Использовала `@react-navigation/native` и `@react-navigation/native-stack`.
* **Функционал:** Создала стек экранов, где с главного экрана можно перейти в "Детали" и вернуться назад с помощью кнопки или системного жеста.

## Как запустить проект:
1. Установите зависимости командой: `npm install`.
2. Установите Expo Go на свой смартфон.
3. Запустите проект: `npx expo start`.
4. Отсканируйте QR-код через приложение Expo Go.

## Ключевые выводы:
* Мобильная разработка сильно отличается от веба отсутствием стандартных HTML-тегов.
* Expo значительно упрощает процесс тестирования приложения на реальном устройстве.
* Верстка на Flexbox в мобильных приложениях требует внимательного отношения к размерам экрана.
