# Лабораторная работа 4.1: Форма регистрации с валидацией

## Подход к обработке событий
В данном проекте реализована форма регистрации пользователя с использованием паттерна **управляемых компонентов**. Состояние каждого поля ввода (имя, email, возраст) синхронизировано с состоянием React через хук `useState`.

## Ключевые особенности:
Синтетические события: Для обработки ввода данных используются объекты событий React (`onChange`), что обеспечивает кроссбраузерную совместимость.
Предотвращение стандартного поведения: В обработчике `onSubmit` вызывается метод `e.preventDefault()`, чтобы предотвратить перезагрузку страницы при отправке формы.
Валидация в реальном времени: Проверка данных (минимум 2 символа для имени, формат email и возраст 18+) происходит мгновенно при изменении текста в поле.
Условный рендеринг: Сообщения об ошибках и сообщение об успешной регистрации отображаются только при наличии соответствующих условий в состоянии.
