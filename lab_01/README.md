## как запустить?

1. зайти в папку проекта: 'cd task2'
2. установка: 'npm install'
3. запустить сайт: 'npm run dev'

Можно просто поменять данные (стейт), и React сам все обновит (декларативный подход). Это бвстро изза Virtual DOM. Верстка с jQuery (кнопка красит текст), React-приложение с такой же кнопкой и счетчиком
