# Лабораторная работа №13: Навигация в мобильных приложениях


## Описание работы
В этой лабораторной я глубоко погрузилась в навигацию. Теперь моё приложение — это не просто один экран, а полноценная система с переходами и нижним меню.

## Что я реализовала:

### 1. Передача данных между экранами (Task 1)
Я научилась не просто открывать новый экран, но и передавать на него нужную информацию.
* **Как это работает:** На главном экране я создала объект с данными пользователя, а при переходе передала его через параметры маршрута.
* **Результат:** Экран деталей «поймал» эти данные через `route.params` и отобразил их. Это очень удобно для создания профилей или страниц товаров.

### 2. Нижнее меню — Tabs (Task 2)
Я создала классическое меню (Bottom Tab Navigation), которое мы видим почти в каждом приложении (как в Instagram или Telegram).
* **Что использовала:** Библиотеку `@react-navigation/bottom-tabs`.
* **Функционал:** Теперь можно быстро переключаться между вкладками "Главная" и "Настройки", причем состояние каждого экрана сохраняется.

## Чему я научилась:
- Работать со стеком навигации (`Stack Navigator`).
- Правильно передавать ID и имена между страницами, чтобы приложение знало, чей профиль открывать.
- Настраивать внешний вид вкладок (иконки, цвета активных элементов).
