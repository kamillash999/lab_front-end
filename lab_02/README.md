# Lab 02: JSX, Fragments & Expressions

Эта лабораторная работа дпоказывает навыки работы с React JSX, фрагментами и динамическими выражениями
Содержит реализацию Lab 2.1 (Фрагменты и списки)
Содержит реализацию Lab 2.2 (Custom компоненты и пропсы)
Запуск проекта
1. Склонируйте репозиторий.
2. Выполните команду `npm install`.
3. Запустите проект командой `npm run dev`.

## Описание компонентов
FragmentLayout: Использует `<>...</>` для группировки header, main и footer без лишних div
ItemList: Отображает массив объектов через `.map()` с использованием уникальных ключей
CombinedFragment: Демонтирует использование `.length` внутри JSX выражения

Card: Компонент-обертка, использующий `children` и динамические классы
ProductList: Маппинг массива продуктов в компоненты Card
Section: Группировка контента через фрагменты с передачей заголовка через пропсы
